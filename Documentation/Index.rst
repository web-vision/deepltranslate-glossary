This extension is a :guilabel:`PUBLIC` `EXT:deepltranslate_core <https://extensions.typo3.org/extension/deepltranslate_core>`_
addon providing the ability to manage `DeepL API glossaries` used to translate
pages, content and records in TYPO3 for languages supported by `DeepL <https://www.deepl.com/de/docs-api/>`_.
